Type Description Example
button defines a button
checkbox defines a checkbox
color used for input fields that should contain a color.
date used for input fields that should contain a date.
datetime-local specifies a date and time input field, with no timezone.
email used for input fields that should contain an email address.
file define a file-select field and a "Browse" button for file downloads.
hidden define a hidden input field.
image define an image as a submit button.
month allow the user to select a month and year.
number define a numeric input field.
password defines a password field.
radio defines a radio button
range define a control for entering a number whose exact value is not important. Default is 0 to 100
reset defines a reset button that will reset all form values to their default values
search used for search field.
submit defines a button for submitting form data to a form-handler.
tel used for input fields that should contain a telephone number.
text defines a one-line text input field
time allow user to select a time.
url used for input fields that should contain a url address.
week allow user to select a week and year.
